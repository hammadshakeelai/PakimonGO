# Tools

Automation and repository support scripts live here.

Planned tools:

- `graphify`: code-graph export and impact reports after code exists.
- `okf-export`: compact knowledge generation from docs and code metadata.
- `qa`: test dataset checks, privacy leak checks, and docs validation.
- `scripts`: small maintenance scripts.

Every non-trivial tool should have a README, dry-run mode, and clear inputs/outputs.
