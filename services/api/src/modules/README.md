# API Modules

Planned modules:

- `auth`: token verification, provider linking, session/account risk.
- `accounts`: profiles, consent, privacy settings, deletion/export.
- `media`: signed uploads, media metadata, derivatives, EXIF policy.
- `submissions`: capture submissions, drafts, observation state.
- `scoring`: score events, explanations, appeals, rollbacks.
- `geo`: PostGIS regions, map cells, geofences, privacy transforms.
- `social`: posts, comments, likes, reposts, groups, hashtags, blocks.
- `leaderboards`: global, country, local, and friends rankings.
- `moderation`: reports, cases, actions, appeals, audits.
- `notifications`: push tokens, preferences, event notifications.
- `taxa`: taxonomy search/display hints and versioned taxon metadata.

Add a module README before adding non-trivial code.
