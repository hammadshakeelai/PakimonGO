# Goldsets

Goldsets are controlled benchmark datasets for AI/scoring quality.

Planned sets:

- `duplicate-detection`: exact, perceptual, crop, screenshot, filtered, and compressed variants.
- `zoo-detection`: inside, near-boundary, uncertain, honest-zoo, and false-zoo cases.
- `species-identification`: common, rare, blurry, partial, local, pet, and no-animal samples.
- `scoring-calibration`: expected score ranges and fairness cases.
- `sensitive-species`: location suppression and policy behavior fixtures.

Use synthetic or properly licensed data first. Store large/private assets outside Git and reference them by manifest.
